## What's changed

**Fixed**
- Resolved an issue where the "Near cursor" overlay occasionally appeared at a fallback position on the first dictation after a model load.
- Fixed a UI bug where the overlay would freeze on the green checkmark (or disappear unexpectedly) if a new dictation was started very rapidly (within 300ms) after stopping the previous one.
- Fixed an issue where the recording processing state wouldn't correctly clear when rejecting very short audio segments.

**Improved**
- Unified the "Check for Updates" button behavior on the About page to match the visual style of the sidebar when a new update is available.
